# basic-syntax

C# の基本構文を確認するための TIL 用サンプルです。

- 条件分岐（if / else）
- ループ（for）
- メソッド定義と呼び出し
- クラスとオブジェクト（Person クラス）
- 文字列補間（$"{...}"）

## ファイル構成

- `BasicSyntaxSample.csproj`  
  .NET プロジェクト設定ファイル

- `Program.cs`  
  メインの C# コード（エントリーポイント）

- `Person.cs`  
  Person クラス（別ファイルに分割）

## 実行方法

```bash
# ディレクトリに移動
cd til/csharp/basic-syntax

# 実行（ビルドも兼ねる）
dotnet run
```
