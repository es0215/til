# basic-syntax

JavaScript での超シンプルな TIL 用サンプルです。

- アラートを表示する
- 変数を宣言して計算する
- 条件分岐（if / else）
- ループ（for）
- 関数を定義して呼び出す
- 配列を操作する
- オブジェクトを作成してプロパティを参照する

## ファイル構成

- `index.html`  
  ブラウザで開くための HTML。`main.js` を読み込みます。

- `main.js`  
  JavaScript の基本構文をまとめたサンプルコードです。

## 実行方法

```bash
# ディレクトリに移動
cd til/javascript/basic-syntax
```

その後、index.html をブラウザで開きます。
（ダブルクリック、または VS Code の Live Server など）

ブラウザの開発者ツール（F12）を開き、「コンソール」タブでログを確認してください。
